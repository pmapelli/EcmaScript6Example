# EcmaScript6Example
Exemplo de uma aplicação desenvolvida para aplicar conceitos do ES6

# Conceitos aplicados:
- Babel
- Classes
- Const & Let
- Operaçes em array
- Arrow Functions
- Valores padrão
- Desestruturação
- Operadores rest/spread
- Template Literals
- Object Short Syntax
- Async/Await
- Axios
